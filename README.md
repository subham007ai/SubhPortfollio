# Subham Sarangi Portfolio

![Subham Sarangi Portfolio](https://subhamsarangi.vercel.app/og-image.png)

A personal portfolio website for Subham Sarangi, an AI Engineering Student (B.Tech CS-AIML) with a focus on Python, Machine Learning, Automation, and building real-world projects.

## Features

- **Modern & Dynamic UI**: Built with Next.js 14, React 18, and Tailwind CSS.
- **Performant Animations**: Framer Motion for smooth scroll and micro-interactions.
- **Fully Responsive**: Optimized for desktop, tablet, and mobile viewing.
- **SEO Optimized**: Includes Metadata, Open Graph tags, Sitemap, and Robots.txt.
- **Custom Cursor & Page Transitions**: Enhances user experience and engagement.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## Local Development

1. **Clone the repository:**
   ```bash
   git clone https://github.com/subham007ai/SubhPortfollio.git
   ```

2. **Navigate into the directory:**
   ```bash
   cd SubhPortfollio
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build and Production

To build the application for production, run:

```bash
npm run build
```

Start the production server:

```bash
npm start
```
